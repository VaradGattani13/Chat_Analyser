# Chat Analyser
A web application based on Streamlit app which gives complete analysis of WhatsApp chat (based on 24 HR Format).<br/>
This includes heatmap,bar graphs, pie charts for clear and clean preview and understanding.<br/>
Primary features of the application includes:<br/>

**Most Active User**  (Both Group and Personal Chat)<br/>

**Most used Words And Emojis**<br/>

*Monthly and Daily Timeline *** (Both Group and Personal Chat)<br/>
**Activity Tracker** <br/>
**Word Cloud** <br/>
